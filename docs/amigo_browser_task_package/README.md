# Amigo Object Browser Mockups

This directory contains mockups and notes for the new egui-based Amigo Object Browser.

Folder convention:

```txt
<window-id>/
  <window-id>.svg
  README.md
```

Each browser window or dialog should have its own folder. Keep SVG mockups, notes, and implementation references for that window together.

Current windows:

```txt
start-dialog/
  start-dialog.svg
```
