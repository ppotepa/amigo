- [x] Planned

## Intent

Usunąć z engine wszystkie pozostałe legacy ścieżki, aliasy i nazwy związane z `platformer`, tak aby:

1. engine znał wyłącznie neutralną domenę `motion_2d`,
2. `playground-sidescroller` był tylko modem / sample gameplayowym,
3. nie istniały już compatibility pathy typu `Platformer*`, `PlatformerController2D`, `amigo-2d-platformer`, `platformer_2d`, `world.platformer`.
