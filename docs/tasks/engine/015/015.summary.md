# TASK 015: playground-2d-particles Showcase + Editor v0

## Cel

Dodać developerski mod `playground-2d-particles`, który jest konsumentem `ParticleEmitter2D`, `Value Curves` i UI controls z taska `014`.

Mod ma być laboratorium particle systemu:

- galeria presetów;
- menu;
- podstawowy editor parametrów;
- runtime mutation API dla emitterów;
- docelowo baza pod pełny particle editor.

## Zależność

Ten task zależy od `014`, bo editor potrzebuje:

- runtime themes;
- slider/toggle;
- `on_change`;
- sensownych paneli UI.

Bez tego editor byłby protezą z przyciskami `+/-`.

## Zakres

Struktura:

```text
mods/playground-2d-particles/
  mod.yml
  scenes/menu/scene.yml
  scenes/menu/scene.rhai
  scenes/showcase/scene.yml
  scenes/showcase/scene.rhai
  scenes/editor/scene.yml
  scenes/editor/scene.rhai
```

Showcase presets:

- Fire
- Smoke
- Sparks
- Magic/Energy
- Snow
- Dust
- Thruster

Editor v0:

- wybór presetu;
- suwaki dla spawn rate, lifetime, speed, spread, initial/final size;
- color preset buttons;
- intensity toggle/slider;
- reset/randomize;
- print YAML do dev console.

## Engine additions

Do `Particle2dSceneService`:

- `set_spawn_rate`
- `set_lifetime`
- `set_speed`
- `set_spread_degrees`
- `set_initial_size`
- `set_final_size`
- `set_color_rgba`
- opcjonalnie `set_shape`

Do Rhai:

- `world.particles.set_spawn_rate(...)`
- `world.particles.set_lifetime(...)`
- `world.particles.set_speed(...)`
- `world.particles.set_spread_degrees(...)`
- `world.particles.set_initial_size(...)`
- `world.particles.set_final_size(...)`
- `world.particles.set_color_rgba(...)`

## Definition of Done

- `playground-2d-particles/menu` bootstrappuje się.
- `showcase` pokazuje co najmniej 7 efektów.
- Każdy efekt używa `ParticleEmitter2D`, bez game-specific engine code.
- Editor może realnie zmieniać podstawowe parametry aktywnego emitera.
- Theme z `014` działa w UI editorze.
- Testy potwierdzają bootstrap menu/showcase/editor i minimum jedną mutację emitera przez Rhai.
