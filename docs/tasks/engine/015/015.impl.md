- `sparks-emitter`
- `magic-emitter`
- `snow-emitter-*`
- `dust-emitter`
- `thruster-emitter`

UI:

- selected preset;
- intensity slider;
- start/stop all buttons;
- per-preset badges.

Rhai:

- `1..7` select preset;
- `Space` toggle selected;
- `A` start all;
- `S` stop all;
- `R` reset intensity;
- `Esc` menu.

## 4. Particle mutation API

```rust

set_initial_size(entity, f32)
set_final_size(entity, f32)
```
