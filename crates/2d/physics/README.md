# amigo-2d-physics

2D collision and kinematic simulation.

## Responsibility
- Kinematic bodies.
- AABB, circle, static collider, and trigger state.
- Overlap queries and collision event rules.
- Scene command adapters for physics data.

## Not here
- Rendering.
- Character controller policy.
- Platform input.

## Depends on
- amigo-core.
- amigo-math.
- amigo-runtime.
- amigo-capabilities.
- amigo-scene.
