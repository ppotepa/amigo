# amigo-2d-vector

2D vector shape scene services.

## Responsibility
- Store line and polygon commands.
- Support debug, gameplay, and lightweight vector rendering.
- Hydrate vector scene commands into runtime state.

## Not here
- GPU rendering.
- Physics collision.
- Complex vector asset editing.

## Depends on
- amigo-core.
- amigo-math.
- amigo-runtime.
- amigo-capabilities.
- amigo-scene.
