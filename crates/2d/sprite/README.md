# amigo-2d-sprite

2D sprite scene services and commands.

## Responsibility
- Store sprite render state from scene documents.
- Expose sprite mutations for runtime systems and scripts.
- Provide sprite draw commands for render extraction.

## Not here
- Texture loading policy.
- GPU rendering.
- Animation graph logic.

## Depends on
- amigo-assets.
- amigo-core.
- amigo-math.
- amigo-render-api.
- amigo-capabilities.
- amigo-runtime.
- amigo-scene.
