# amigo-2d-motion

2D movement controllers and projectile helpers.

## Responsibility
- Platformer-style motion controller state.
- Freeflight motion.
- Velocity stepping.
- Bounds handling.
- Projectile emitter launch helpers.

## Not here
- Collision resolution.
- Input mapping.
- Rendering.

## Depends on
- amigo-2d-physics.
- amigo-core.
- amigo-math.
- amigo-runtime.
- amigo-capabilities.
- amigo-scene.
