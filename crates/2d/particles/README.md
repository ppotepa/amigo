# amigo-2d-particles

2D particle runtime and authored emitter services.

## Responsibility
- Particle emitter runtime state.
- Spawn areas, forces, curves, shapes, lights, and material flags.
- Runtime particle controls for scripts and tools.
- Export emitter config back to scene YAML.

## Not here
- GPU particle rendering.
- General FX primitives.
- Asset catalog ownership.

## Depends on
- amigo-core.
- amigo-fx.
- amigo-math.
- amigo-capabilities.
- amigo-runtime.
- amigo-scene.
