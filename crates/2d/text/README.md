# amigo-2d-text

2D text scene service for labels and HUD-style content.

## Responsibility
- Queue world-space 2D text commands.
- Store text content, font asset, bounds, and transform.
- Register 2D text capability.

## Not here
- Font rasterization.
- UI layout.
- Renderer implementation.

## Depends on
- amigo-assets.
- amigo-core.
- amigo-math.
- amigo-render-api.
- amigo-capabilities.
- amigo-runtime.
- amigo-scene.
