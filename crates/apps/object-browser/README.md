# Amigo Object Browser

egui-based tooling app for browsing Amigo mods.

Current scope:

```txt
- scan mods/*/mod.toml
- show a start dialog with mod summaries
- select a mod
- show scene list and content counts
- expose placeholder Open actions
```

Run:

```txt
cargo run -p amigo-object-browser
```

Optional:

```txt
cargo run -p amigo-object-browser -- --mods-root mods
```
