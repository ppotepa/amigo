# amigo-audio-generated

Generated and procedural audio authoring utilities.

## Responsibility
- Parse generated audio assets.
- Define tone, envelope, waveform, and parameter models.
- Produce sample data for mixer/output use.

## Not here
- Runtime playback queue.
- Final mixdown.
- Device output.

## Depends on
- amigo-assets.
- amigo-core.
- amigo-runtime.
- amigo-capabilities.
- serde_yaml.
