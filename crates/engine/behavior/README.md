# amigo-behavior

Behavior graph and state orchestration for authored gameplay logic.

## Responsibility
- Behavior definitions.
- Runtime behavior state.
- Shared structures for behavior-driven systems.

## Not here
- Scripting backend bindings.
- Physics or rendering domains.
- Platform input handling.

## Depends on
- amigo-core.
- amigo-fx.
- amigo-math.
- amigo-runtime.
