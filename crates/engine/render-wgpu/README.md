# amigo-render-wgpu

WGPU-based renderer and overlay implementation.

## Responsibility
- GPU setup.
- 2D, 3D, tilemap, particle, UI, and overlay rendering.
- Render backend integration with the engine render API.

## Not here
- Scene parsing.
- Platform window creation.
- Gameplay simulation.

## Depends on
- amigo-render-api.
- amigo-window-api.
- amigo-assets.
- 2D, 3D, UI, and math crates.
- wgpu.
