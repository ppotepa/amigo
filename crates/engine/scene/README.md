# amigo-scene

Scene document, hydration, and command pipeline.

## Responsibility
- Scene file model.
- Scene command formats.
- Entity metadata and lifecycle state.
- Hydration from authored scene data into runtime commands.

## Not here
- Domain service implementations.
- Rendering.
- Platform IO.

## Depends on
- amigo-assets.
- amigo-core.
- amigo-fx.
- amigo-math.
- amigo-runtime.
- serde.
- serde_yaml.
