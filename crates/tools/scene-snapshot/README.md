# amigo-tool-scene-snapshot

Scene snapshot service crate.

## Responsibility
- Snapshot request and image models.
- Placeholder snapshot service.
- Runtime-backed scene snapshot service.
- File snapshot service.

## Not here
- Interactive preview UI.
- Editor frontend state.
- Renderer implementation details beyond snapshot output.

## Depends on
- amigo-app.
- amigo-modding.
- image.
- serde_yaml.
