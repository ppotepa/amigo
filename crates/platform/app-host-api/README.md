# amigo-app-host-api

Host lifecycle API for driving an Amigo runtime.

## Responsibility
- Host configuration.
- Lifecycle events.
- Host control flow.
- Host handler trait used by app shells.

## Not here
- Winit implementation.
- Rendering backend.
- Engine simulation.

## Depends on
- amigo-core.
- amigo-input-api.
- amigo-window-api.
