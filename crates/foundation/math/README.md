# amigo-math

Low-level math primitives shared by all engine domains.

## Responsibility
- Vectors, transforms, colors, and scalar curves.
- Small math types used by rendering, scene, UI, FX, and gameplay crates.
- Dependency-light data structures.

## Not here
- Rendering code.
- Physics simulation.
- Engine services or runtime state.

## Depends on
- Nothing inside the workspace.
