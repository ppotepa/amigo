# amigo-3d-text

3D text scene service for world-space labels.

## Responsibility
- Store text content, font asset, size, and transform.
- Queue 3D text draw commands.
- Register 3D text capability.

## Not here
- Font rasterization.
- UI layout.
- Renderer implementation.

## Depends on
- amigo-assets.
- amigo-math.
- amigo-capabilities.
- amigo-runtime.
- amigo-scene.
