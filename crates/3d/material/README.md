# amigo-3d-material

3D material scene service for mesh rendering.

## Responsibility
- Store material labels.
- Store albedo and optional texture source.
- Bind material parameters to entities.

## Not here
- Mesh ownership.
- Shader implementation.
- Texture loading policy.

## Depends on
- amigo-assets.
- amigo-core.
- amigo-math.
- amigo-render-api.
- amigo-capabilities.
- amigo-runtime.
- amigo-scene.
