# amigo-ui

Runtime UI document, layout, and state services.

## Responsibility
- UI document model.
- Layout computation.
- Themes, bindings, tabs, events, and input state.
- Scene bridge for authored UI data.

## Not here
- Native desktop widgets.
- Renderer-specific draw implementation.
- Editor-only UI code.

## Depends on
- amigo-assets.
- amigo-core.
- amigo-math.
- amigo-capabilities.
- amigo-runtime.
- amigo-scene.
